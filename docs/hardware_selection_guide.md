## 1. 心率监测设备

| ⭐⭐⭐ | **Polar H10** | 博能 | ¥699-799 | 京东自营 | 专业级精度，支持 BLE + ANT+，可存储训练数据 |
| ⭐⭐⭐ | **Polar H9** | 博能 | ¥499-590 | 京东/淘宝 | 性价比之选，精度与 H10 相当，无数据存储 |
| ⭐⭐ | **CooSpo H9Z** | CooSpo | ¥169-269 | 淘宝 | 国产高性价比，支持 BLE + ANT+ |
| ⭐⭐ | **CooSpo HW706** | CooSpo | ¥199-299 | 京东/淘宝 | 臂带式，佩戴更舒适 |
| ⭐ | **Garmin HRM-Dual** | 佳明 | ¥399-499 | 京东自营 | 双模传输，兼容性好 |

### 购买链接参考

**京东搜索关键词：**
- `Polar H10 心率带`

| 场景 | 推荐产品 | 理由 |

```yaml
# config/devices.yaml 配置示例
lights:
  devices:
    # Yeelight 配置
    - name: main_light
      type: yeelight
      config:
        ip: "192.168.1.100"  # 需要在路由器中查看
    
    # 飞利浦 Hue 配置（需要桥接器）
    - name: hue_light
      type: hue
      config:
        bridge_ip: "192.168.1.50"
        api_key: "your-api-key"  # 通过桥接器注册获取
        light_id: "1"
```

```yaml

  type: wifi  # 或 tuya

    port: 80
  # 涂鸦设备配置
  tuya:
```

1. 注册涂鸦 IoT 平台账号：https://iot.tuya.com/
2. 创建云项目，关联涂鸦智能APP
3. 在设备管理中获取设备信息

## 附录：配置文件模板

根据您选购的设备，修改 `config/devices.yaml` 配置文件：

```yaml
# 全局设置
global:
  connection_timeout: 10
  enable_mock_fallback: true  # 设备未连接时使用模拟模式

# 心率设备
heart_rate:
  enabled: true
  type: ble
  ble:
    device_name_keywords:
      - "Polar"  # 或 "CooSpo"
      - "H10"    # 或 "H9", "H9Z"

# 灯光设备
lights:
  enabled: true
  devices:
    - name: main_light
      type: yeelight  # 或 hue, mihome, tuya
      config:
        ip: "192.168.1.100"  # 替换为实际IP

# 座椅
chair:
  enabled: true
  type: mock  # 大多数按摩设备需要使用模拟模式

# 香薰
scent:
  enabled: true
  type: mock  # 或 wifi, tuya
  wifi:
    ip: "192.168.1.110"

# 音频
audio:
  enabled: true
  default_volume: 50
```

## 购买注意事项

1. **确认网络协议**：购买前确认设备支持 WiFi 局域网控制或 BLE 蓝牙
2. **查看开发文档**：优先选择有开放 API 或 SDK 的产品
3. **固定 IP 地址**：WiFi 设备建议在路由器中设置静态 IP
4. **保留购买凭证**：方便退换货和售后服务
5. **批量采购**：如需多台部署，可联系商家获取批发价格
